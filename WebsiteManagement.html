Web Site Management - Pre-requisite IT and Web Design *Instructor Approval Offered for all 3 Trimesters

Students will be managing, designing and creating Open to Sophomores, Juniors and Seniors that have taken IT and Web Design

This is not a coding class. We will be using a web-based program to manage a site for the high school. There will be various responsibilities from wire-framing, layout and design to multiple tasks involved in the administration of a Web Site, including hosting, monitoring, verification that all contained links are active and up to date.
